Hulpmiddel

Dit hulpmiddel is een hulpmiddel dat helpt met verschillende dingen.

Mogelijkheden

  • Installeer populaire apss
  • Check het weer van elke stad.
  • Track IP adressen.
  • Maak foto's.
  • Vertaal teksten.
  • Krijg recepten.
  • Verklein urls
  • En nog veel meer...

Installatie

  1. Zorg ervoor dat python en pip al geinstalleerd zijn.
  2. Ga naar deze url: https://github.com/MonAchpro/hulpmiddel
  3. Klik op de code knop en dan op downloaden als zip.
  4. Pak het zip bestand uit.
  5. Ga naar de terminal op de locatie van de uitgepakte map.
  6. Voer dit uit: "pip install -r requirements.txt"
  7. Open het hulpmiddel_simpel bestand en profiteer!

    8. Voor een lijst van mogelijkheden: typ help.

Bewerkbaarheid

Deze code is zo gemaakt zodat alles heel makkelijk te bewerken is.

Om extra sites toe te voegen, voeg je het toe aan "sites.json"

Om extra download links toe te voegen, voeg je het toe aan "urls.json"
